## March Madness Bracket Builder
Every year, I have created a statistics-powered March Madness simulation. This year, I decided to turn this project into a more customizable website available for family and friends. All statistics are proprietary and powered by Kenpom.com.
## Tech Stack:
 - React/Next.js
 - Tailwind CSS
 - Typescript Node.js Backend
